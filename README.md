# pensamento_computacional_projeto-barbearia Gestão de Barbearia - Sistema em Python
Este projeto é um protótipo de sistema de gerenciamento para barbearias, focado na organização de atendimentos e controle de serviços. O software utiliza conceitos de Programação Orientada a Objetos (POO) e manipulação de coleções para gerenciar o fluxo de clientes.
🚀 Funcionalidades
O sistema oferece as seguintes opções no menu principal:
Agendamento de Clientes: Cadastro de nome, escolha do serviço (Corte, Barba ou Combo) e definição de horário.
Visualização de Agenda: Listagem de todos os clientes que já foram registrados no sistema.
Alteração de Serviço: Busca de clientes já agendados para atualização do procedimento desejado.
Cancelamento e Relatórios: Estrutura preparada para remoção de horários e geração de informativos de serviços prestados.
Seleção de Profissional: Atribuição de barbeiros específicos (Felipe, Paulo ou João) para cada atendimento.
🛠️ Tecnologias e Estruturas Utilizadas
O código foi desenvolvido focando na lógica de programação pura, utilizando:
Classes (class): Utilização de Orientação a Objetos para representar a entidade Barbeiro.
Listas ([]): Atuam como banco de dados temporário para armazenar os agendamentos (agenda = []).
Dicionários ({}): Armazenamento estruturado dos dados de cada cliente (nome, serviço e horário).
Laços de Repetição (while): Garantem que o sistema continue rodando até que o usuário escolha sair.
Estruturas Condicionais (if/elif/else): Responsáveis pela validação de opções de menu, tipos de serviço e preços.
